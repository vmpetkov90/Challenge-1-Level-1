# Personal Portfolio Challenge:

## Challenge: 
Create a personal portfolio webpage showcasing your skills, projects, and contact information using HTML and CSS.

## Requirements:
Include a navigation menu, sections for About Me, Projects, Contact, and any other relevant information. Use CSS to style the layout and design.
